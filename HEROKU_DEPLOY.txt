STEP 1: INSTALL HEROKU CLI
================================================================================

Option A: Using Homebrew (if installed)
  brew tap heroku/brew
  brew install heroku

Option B: Direct Download
  https://devcenter.heroku.com/articles/heroku-cli
  Download macOS installer and run

Option C: Verify Installation
  heroku --version

================================================================================
STEP 2: LOGIN TO HEROKU
================================================================================

  heroku login
  # Opens browser to authenticate
  # Sign in or create free account (heroku.com)

Verify login:
  heroku auth:whoami

================================================================================
STEP 3: CREATE HEROKU APP
================================================================================

  cd /Users/philippe/Documents/Claude/bus_ma
  heroku create bus-ma-targets

Output will show:
  Creating ⣾ bus-ma-targets... done
  https://bus-ma-targets.herokuapp.com/ | https://git.heroku.com/bus-ma-targets.git

Note: Replace "bus-ma-targets" with unique name if taken

================================================================================
STEP 4: DEPLOY TO HEROKU
================================================================================

Push code to Heroku:
  git push heroku main

Output will show build progress:
  -----> Building on the Heroku platform from buildpack sources
  -----> Using buildpack: heroku/python
  -----> Installing dependencies
  -----> Launching...
  https://bus-ma-targets.herokuapp.com/

================================================================================
STEP 5: OPEN LIVE DASHBOARD
================================================================================

  heroku open
  
OR manually visit:
  https://bus-ma-targets.herokuapp.com/targets

Share this URL with client:
  👉 https://bus-ma-targets.herokuapp.com/targets

================================================================================
OPTIONAL: CONFIGURE DOMAIN
================================================================================

If you want custom domain (e.g., bus-ma.company.com):

  heroku domains:add bus-ma.company.com
  # Update DNS CNAME to d.heroku.com

Full instructions:
  heroku domains:add --help

================================================================================
MONITORING & UPDATES
================================================================================

View logs:

  heroku restart

  heroku config

Push updates:
  git push heroku main  (repeats step 4)

TROUBLESHOOTING
