SCORING METHODOLOGY
================================================================================

M&A Fit Score = Independence × (0.5×Size + 0.3×Clarity + 0.2×Succession)

Component Breakdown:
  • Independence (100%): 1.0 if standalone, 0.2 if in group
  • Size (50%): Facility parking area (m²) as proxy for fleet capacity
  • Clarity (30%): Ownership structure simplicity (single family vs. complex)
  • Succession (20%): Owner age, media signals, transition indicators

Data Quality:
  ✅ FTA roster               — 100% coverage, authoritative
  ✅ Group detection          — 100% coverage (address/email/president)
  ✅ Facility estimates       — 78% coverage, ±30% accuracy
  ⚠️  CTQ permits            — NOT INCLUDED (reCAPTCHA blocks automation)
  ⚠️  Succession signals      — MINIMAL (requires Phase 2 media research)

Example SQL:
  -- Top-30 independents
  SELECT name, city, president, parking_area_m2, ma_fit_score
  FROM scores s
  JOIN operators o ON s.operator_id = o.id
  LEFT JOIN facility f ON o.id = f.operator_id
  WHERE o.group_id IS NULL
  ORDER BY ma_fit_score DESC LIMIT 30;
